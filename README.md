# guia-para-analistas-notebooklm
O guia completo para tornar-se um Analista de Dados e ingressar nessa carreira que abre muitas portas e continua crescendo exponencialmente.
Todo o estudo aqui contido foi feito baseado no NotebookLM do Google, utilizando prompt específico somente para a busca e aprendizado para carreira de Analista de Dados, foi utilizado alguns livros e o Deepsearch para a conclusão da pesquisa.
